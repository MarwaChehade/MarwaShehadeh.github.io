This is The Map, the first free website template from LIU.com which is built upon a CSS Map system. Inspired by the 1KB CSS Map and other systems, The Map template has a 12-column Map based on 60 pixels wide columns with 20 pixels of space inbetween. To use the Map, you create a div with the class = "row" and then place any number of divs with class = "col" and a second class stating the width of the column (c1-c12) inside the row. The Map supports nested rows which means that thousands of layout combinations can be created, from single-column pages to advanced magazine-style layouts. The template uses standards-complaint code.
